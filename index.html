CONHEÇA O SCRATCH

O Scratch é uma plataforma de programação visual desenvolvida pelo MIT para ajudar crianças e iniciantes a aprenderem programação de forma divertida e acessível. Sua interface gráfica é baseada em blocos de código que representam diferentes comandos e operações, os quais são arrastados e soltos para formar scripts. Esses blocos incluem ações como movimento, som, aparência e controle, organizados em categorias para facilitar o acesso

Tetris

Criador: smelly_hobbit

Setas para esquerda e direita para mover, A & S para girar (ou seta para cima), seta para baixo para acelerar. Para jogar novamente pressione a bandeira verde

Faster

Criadores: livcar_su e JuCD

Como jogar? As palavras aparecerão em cores distintas. Cada tecla possui uma cor, então clique na cor de *dentro* da palavra que aparecer na tela. Teremos 3 níveis, então seja rápido e vença seu oponente!

Super Mario for Scratch

Criador: Brad-Games

Use as setas para esquerda/direita para se mover e a seta para cima para pular. Evite tocar inimigos, ou você vai perder vida. Colete cogumelos para ganhar vida e pegue o cogumelo 1UP para ganhar uma vida extra! Pule em cima dos trampolins para saltar alto e não fique nas plataformas quadriculadas por muito tempo, senão você vai cair! Na batalha final, Bowser vai andar para frente e para trás algumas vezes cuspindo bolas de fogo e depois vai abaixar.

Desenvolvido por: Lívia C. Suana e Marina Fonseca
